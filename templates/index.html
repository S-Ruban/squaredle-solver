About

Squaredle is a word puzzle where you have to find words in a grid by visiting each cell at most once. It's like crossword, but more flexible.

The grid is solved by using a trie to store all the words in the English dictionary. Then, at every cell, depth-first search (DFS) is performed on the neighbouring elements as long as they are within bounds and not visited yet in that particular DFS call. With every DFS call, the program keeps track of the letters added, and checks if the current string is a prefix in the trie or not. Once it finds out that it has reached the end of a particular word in the trie, it adds that word to the list and continues. The English dictionary is imported from a Python package.
